# Third-party references

- **Alpha²** — Feng Xu et al., *Discovering Logical Formulaic Alphas using Deep Reinforcement Learning*, arXiv:2406.16505. The official `x35f/alpha2` repository contains pseudocode and algorithm structures rather than a runnable implementation. This project provides an original single-asset time-series adaptation and labels the methodological differences in the UI.
- **efinance** — `Micro-sheep/efinance`, used as an optional historical-data adapter. The upstream README limits use to learning and communication and disallows commercial use; confirm upstream terms before any commercial deployment.
- **QuantDinger** — its watchlist, snapshot and audit concepts informed an original lightweight local implementation. No QuantDinger frontend code was copied.
- **Nailong imagery** — derived from the user-supplied references for this local project. Confirm all relevant character/image rights before public or commercial distribution.
